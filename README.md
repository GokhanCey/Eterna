Eterna reimagines NPCs not as static scripts, but as living, evolving entities secured by consensus. By leveraging Amadeus's unique **Consensus-Level Compute**, Eterna agents run their inference deterministically across the network.

When you speak to an Eterna agent, your message is a transaction. The network validates the inference, updates the agent's memory state (Merkle Tree), and returns a response that is globally verified.
